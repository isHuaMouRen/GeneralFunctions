# Huazi.Library

一个通用工具库

## 如何使用

- 前往Github仓库的[Release界面](https://github.com/isHuaMouRen/GeneralFunctions/releases)下载 `.dll` 文件，包含在你的项目中

- 命令行或Visual Studio控制台下载[Nuget包](https://www.nuget.org/packages/HuaZi.Library.NET)

*对于.NET:*

```bash
Install-Package HuaZi.Library.NET
```

*对于WinForms*

```bash
Install-Package HuaZi.Library.Forms
```
